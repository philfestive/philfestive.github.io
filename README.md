# PhilFestive

Personal portfolio and link hub for PhilFestive - featuring cozy Godot games and WoW addons.

## Features
- Glassmorphism UI with purple/gold theme
- Animated particle background
- Project roadmap with progress tracking
- Social links and support options
- Fully responsive (mobile-first)
- Accessibility features (reduced motion, keyboard navigation)

## Tech Stack
- Pure HTML/CSS/JavaScript (no build tools)
- CSS Custom Properties for theming
- Canvas-based particle system
- GitHub Pages hosting

## CSS Architecture
The site uses a custom property system for consistency:

### Colors
- `--color-phil` (#9146FF) - Primary purple
- `--color-gold` (#FFD700) - Accent gold
- `--color-text-primary` - Main text
- `--color-text-muted` - Secondary text

### Spacing Scale
- `--space-sm` (8px) → `--space-3xl` (48px)

### Typography Scale
- `--font-size-logo` (42px) → `--font-size-tiny` (10px)

## Development
1. Clone the repo
2. Open `index.html` in browser
3. Edit and refresh

## Accessibility
- Respects `prefers-reduced-motion`
- Keyboard navigable with visible focus states
- Skip-to-content link
- Semantic HTML structure
