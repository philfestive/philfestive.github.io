# UptimeRobot Monitoring Setup for PhilFestive

## Overview

Free uptime monitoring to receive email alerts if any linked platforms go down.
Free tier includes 50 monitors with 5-minute check intervals.

## Step 1: Create Account

Sign up at: https://uptimerobot.com

(No credit card required for free tier)

## Step 2: Add Monitors

For each URL below, click "Add New Monitor" with these settings:
- **Monitor Type:** HTTP(s)
- **Monitoring Interval:** 5 minutes (free tier default)
- **Alert Contact:** Your email
